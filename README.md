## 介绍
`dax1 UI` 是一款基于 Vue 3 和 TypeScript 的 UI 组件库。

`dax1` 取自于胃药“**达喜**”,希望能成为一颗灵药,帮助需要的人,不畏艰难，勇往直前

封装这套组件库的初衷是为了学习,Vue3 和 TypeScript都是第一次在项目应用,还需要积累更多的实战经验

## 安装
打开终端运行下列命令：

```javascript
npm install dax1-ui
```

或

```javascript
yarn add dax1-ui
```

## 开始使用
请先安装本组件库。

然后在代码中写入下面的代码

```javascript
import {Button, Tabs, Switch, Dialog} from "dax1-ui"
```

就可以使用我提供的了
